发布文章介绍

1. 文章配置

按照以下内容格式编写md文件

参数 说明
title 【必填】文章标题
date 【必填】发布日期
updated 【可选】更新日期
tags 【可选】文章标签
categories 【可选】文章分类
keywords 【可选】文章关键字
description 【可选】文章描述
top_img 【可选】文章顶部图片
cover 【可选】文章封面
toc 【可选】是否开启目录
toc_number 【可选】是否开启目录序号
toc_style_simple 【可选】是否开启简单目录
copyright 【可选】显示文章版权模块
copyright_author 【可选】文章版权模块的文章作者
copyright_author_href 【可选】文章版权模块的文章作者链接
copyright_url 【可选】文章版权模块的文章链接
copyright_info 【可选】文章版权模块的文章版权信息
mathjax 【可选】是否开启数学公式支持
katex 【可选】是否开启katex公式支持
aplayer 【可选】是否开启aplayer支持
highlight_shrink 【可选】是否开启代码收缩功能
aside 【可选】是否开启文章侧边栏
abcjs 【可选】加载 abcjs (当设置 abcjs 的 per_page: false 时，才需要配置，默认 false )

之后进行编写文章即可
